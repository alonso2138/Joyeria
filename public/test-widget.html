SaaS Integration Guide v1.0

Visualizador de Joyería IA

Integra la experiencia de probador virtual en tu e-commerce en menos de 5 minutos.

1
Importar el Widget

Añade el siguiente script antes del cierre de la etiqueta </body> en tu página de producto.

<!-- Reemplaza 'tu-dominio.com' por la URL de despliegue -->
<script src="https://tu-dominio.com/widget.js"></script>

2
Configurar los Botones

Cualquier elemento con el atributo data-try-on activará el visualizador. Debes pasar la información del producto mediante atributos data-*.

Anillo Eternity

Collar Diamante

Atributos disponibles:

Atributo Descripción Obligatorio
data-api-key Tu llave de cliente (proporcionada en el panel admin). Sí
data-image URL de la imagen del producto (fondo transparente recomendado). Sí
data-name Nombre del producto que aparecerá en el visualizador. No
data-category Categoría (Anillo, Collar, Reloj, Pulsera). Define la IA a usar. Sí

3
Manejo de Eventos (Opcional)

El widget emite eventos mediante window.postMessage para que puedas capturar el resultado y, por ejemplo, guardarlo en el carrito o enviarlo por email.

window.addEventListener('message', (event) => {
    if (event.data.type === 'TRYON_SUCCESS') {
        console.log('Imagen generada!', event.data.image); // URL base64
        // Aquí puedes realizar acciones personalizadas
    }
});
